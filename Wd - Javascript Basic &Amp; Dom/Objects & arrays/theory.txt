1.) What is an object in JavaScript?


In JavaScript, an object is a structure that stores data in key–value pairs.
The keys are called properties, and if a value is a function, it's called a method.

Example
const car = {
    brand: "Toyota",
    model: "Camry",
    start() {
        console.log("Car started");
    }
};

console.log(car.brand); // "Toyota"
car.start();            // method call


2.) How do you access object properties?

1. Dot Notation
Use a dot (.) followed by the property name.

console.log(car.brand);


2. Bracket Notation
Use square brackets with the property name as a string.

console.log(car["brand"]);



3.) What is an array? Explain array methods.


An array is a data structure used to store multiple values in a single variable. These values
are stored in contiguous (continuous) memory locations and can be accessed using an index.
Most programming languages start array indexing at 0, meaning the first element is at index 0.


Array methods are built-in functions that let you:

1.Add or remove items
2.Search for items
3.Loop through items
4.Transform the array
5.Sort or combine arrays

They make working with arrays easier and more powerful.

Useful Array Methods 
1. Add / Remove Items
push() – add to the end
arr.push(4);
pop() – remove from the end
arr.pop();
unshift() – add to the beginning
arr.unshift(0);
shift() – remove from the beginning
arr.shift();

2. Search / Check
indexOf() – find the position
arr.indexOf("banana");
includes() – check if item exists
arr.includes("apple");
find() – find the first item that matches a condition
arr.find(n => n > 10);

3. Transform / Filter
map() – create a new array by modifying each item
arr.map(n => n * 2);
filter() – keep only items that match a condition
arr.filter(n => n % 2 === 0);
reduce() – combine items into one value
arr.reduce((sum, n) => sum + n, 0);

4. Loop Through Items
forEach()
arr.forEach(item => console.log(item));

5. Copy / Combine / Modify
slice() – copy part of an array
arr.slice(1, 3);
splice() – add/remove items (changes original)
arr.splice(2, 1);
concat() – join two arrays
arr.concat([4, 5]);

6. Sorting
sort()
arr.sort();
reverse()
arr.reverse();



4.) Explain push(), pop(), slice(), sort(), and filter().


1.) push()

Adds one or more elements to the end of an array.
Changes (mutates) the original array.

Example:
let arr = [1, 2, 3];
arr.push(4);  
// arr becomes [1, 2, 3, 4]

2.) pop()

Removes the last element of an array
Also changes the original array and returns the removed value.

Example:
let arr = [1, 2, 3];
let removed = arr.pop();  
// removed = 3
// arr becomes [1, 2]

3.) slice()

Copies a portion of an array and returns a new array.
❗ Does not change the original array.

Syntax:

slice(start, end) → end is not included

Example:
let arr = [10, 20, 30, 40, 50];
let part = arr.slice(1, 4);  
// part = [20, 30, 40]
// original arr stays the same

4.) sort()

Sorts elements inside the array.
It changes the original array.

Example (default: alphabetical order):
let arr = [3, 1, 5, 2];
arr.sort();  
// arr becomes [1, 2, 3, 5]
Example (numerical sorting with compare function):
arr.sort((a, b) => a - b);  

5.) filter()

Creates a new array containing only items that pass a condition.
Does not change the original array.

Example:
let arr = [1, 2, 3, 4, 5];
let even = arr.filter(n => n % 2 === 0);
// even = [2, 4]
// arr stays the same
